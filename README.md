# Endless Jumper for Reinforcement Learning

The purpose of this project is to create a simulation environment for an Icy Tower-inspired game. The game is designed to mimic the core behaviors of the original, including:

- character acceleration and movement
- jump height calculation based on momentum
- wall bounce physics
- score tracking with combos
- screen scrolling dependent on elapsed time

The goal is to train a reinforcement learning agent to maximize the score, comparing performance across several different policies.
